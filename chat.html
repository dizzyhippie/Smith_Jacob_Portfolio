Project

The goal of the project was to create an interactive chat application using Express, Socket.io, and Vue. This app allowed users to connect to a server and send messages back and forth. Either anonymously or with a username chosen by them.

Process

Some of the main features include: typing notifications, new message alert sound, and a side panel showing who is currently in the chat. Some extra funtionality was added to display messages like a text conversation.

Outcome

The app functioned well and used a lot of the features available with socket.io. I plan to add additional functionality in the future to allow private messaging, nicknames and more!
